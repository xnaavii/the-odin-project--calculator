# Calculator

A JavaScript calculator built as part of learning DOM manipulation and event handling.

[Live Demo](#) · [Source Code](#)

## Features

- Basic arithmetic — add, subtract, multiply, divide
- Operator chaining (e.g. `1 + 2 - 3`)
- Keyboard support alongside click
- Backspace to correct input
- Division by zero handling

## What I learned

- Managing state with plain JS variables and keeping types consistent
- Abstracting input handling so click and keyboard share the same logic
- Edge cases in calculator UX — chaining operators, clearing after a result

## Stack

Vanilla JavaScript · HTML · CSS
